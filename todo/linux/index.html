Root远程登录设置

#!/bin/bash
# setup root access - default login: oom/oom - comment out to restrict access too ssh key only
sed -i 's/PermitRootLogin.*/PermitRootLogin yes/' /etc/ssh/sshd_config
sed -i 's/PasswordAuthentication.*/PasswordAuthentication yes/' /etc/ssh/sshd_config
service sshd restart
echo -e "mano2018\nmano2018" | passwd root

IP设置

16.04不介绍了，这里主要说明18.04后的差异。 18.04后，ubuntu的IP设置修改/etc/network/interfaces不再生效，需要参考如下方式修改IP

配置IP

sudo vi /etc/netplan/*.yaml
network:
    ethernetes:
        ens160:
            address: [10.190.49.26/22]
            gateway4: 10.190.48.1
            dhcp4: false
            optional: true
    version: 2
sudo netplan apply

DNS配置

sudo vi /etc/resolv.conf
nameservice 10.19.8.15

系统软件源镜像

公司搭建的18.04的官方仓库源镜像，需要手动编辑/etc/apt/sources.list 注释或删除其中与以下配置相似的，替换为镜像站地址

deb http://cn.archive.ubuntu.com/ubuntu/ bionic main universe multiverse restricted
deb http://cn.archive.ubuntu.com/ubuntu/ bionic-updates main universe multiverse restricted
deb http://security.ubuntu.com/ubuntu/ bionic-security main universe multiverse restricted
deb http://security.ubuntu.com/ubuntu/ bionic-backports main universe multiverse restricted
