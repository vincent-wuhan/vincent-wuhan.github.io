Linux常用技巧

Mar 1, 2020
1 min read

  • 查看磁盘大文件

    du -sh /*

  • 修改网卡名字（如eth0）

    /etc/udev/rules.d/70-persistent-net.rules

  • Root远程登录设置

    #!/bin/bash
# setup root access - default login: oom/oom - comment out to restrict access too ssh key only
sed -i 's/PermitRootLogin.*/PermitRootLogin yes/' /etc/ssh/sshd_config
sed -i 's/PasswordAuthentication.*/PasswordAuthentication yes/' /etc/ssh/sshd_config
service sshd restart
echo -e "mano2018\nmano2018" | passwd root

  • Root用户密码设置

    sudo passwd root

  • 查看apt-get依赖包

cd /var/cache/apt/archives

  • 添加用户到docker组

root@mano:~# gpasswd -a sdc docker Adding user sdc to group docker root@mano:~# newgrp docker

  • IP设置

16.04不介绍了，这里主要说明18.04后的差异。 18.04后，ubuntu的IP设置修改/etc/network/interfaces不再生效，需要参考如下方式修改IP

配置IP

sudo vi /etc/netplan/*.yaml

network:
    ethernetes:
        ens160:
            address: [10.190.49.26/22]
            gateway4: 10.190.48.1
            dhcp4: false
            optional: true
    version: 2

sudo netplan apply

DNS配置

sudo vi /etc/resolv.conf

nameservice 10.19.8.15
  • 修改软件源镜像

手动编辑/etc/apt/sources.list

  • 目录下查找文件

    find DIR_PATH -name "FILE_NAME"

  • 软链接

ln -s 源目录 目的目录

例如: 将/home/mano/installRel的快捷方式放到/opt/file/installRel中

ln -s /home/mano/installRel /opt/file/installRel

  • 代理设置

    export http_proxy="http://10.190.51.201:7788"  export https_proxy="http://10.190.51.201:7788"

  • 查看tar包目录结构

    gzip -dc *.tar.gz |tar tvf -

  • awk查询获取除第一行外的其他行

以获取docker运行的容器id为例

docker ps|awk 'NR == 1 {next} {print $1}'

NR == 1指不打印第一行
